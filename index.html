Métodos de Arrays y Strings en JavaScript

Método Descripción Ejemplo Resultado
map() Crea un nuevo arreglo con los resultados de aplicar una función a cada elemento. [1,2,3].map(n => n*2) [2,4,6]
filter() Devuelve los elementos que cumplen una condición. [1,2,3,4,5].filter(n => n%2===0) [2,4]
reduce() Reduce el arreglo a un solo valor. [1,2,3,4].reduce((a,b)=>a+b) 10
find() Devuelve el primer elemento que cumple una condición. [5,12,8].find(n=>n>10) 12
sort() Ordena los elementos del arreglo. [10,2,30].sort((a,b)=>a-b) [2,10,30]
includes() Verifica si un valor existe en el arreglo. ["manzana","pera"].includes("pera") true
indexOf() Devuelve el índice del elemento. ["a","b","c"].indexOf("b") 1
some() Verifica si al menos un elemento cumple una condición. [1,2,3].some(n=>n>2) true
every() Verifica si todos los elementos cumplen una condición. [2,4,6].every(n=>n%2===0) true
slice() Copia una parte del arreglo sin modificarlo. ["a","b","c","d"].slice(1,3) ["b","c"]
splice() Agrega, elimina o reemplaza elementos (modifica el original). ["a","b","c"].splice(1,1,"x") ["a","x","c"]
join() Une los elementos del arreglo en una cadena. ["Hola","mundo"].join(" ") "Hola mundo"
split() Divide una cadena en un arreglo según un separador. "Hola mundo feliz".split(" ") ["Hola","mundo","feliz"]
